# Слайд 1: Введение в Symbol

## Проблема коллизий ключей в объектах

В JavaScript объекты используют строки в качестве ключей. Это может привести к конфликтам, когда разные части кода используют одни и те же имена ключей.

```javascript
const obj = { id: 1 };
obj.id = 2; // перезаписали значение
```

## Ограничения строковых ключей

Строковые ключи не гарантируют уникальность, что может привести к непредсказуемому поведению при совместной работе с объектами.

## Исторический контекст (ES6)

Symbol был введен в ECMAScript 2015 (ES6) как способ создать уникальные идентификаторы для свойств объектов, чтобы избежать коллизий ключей.
