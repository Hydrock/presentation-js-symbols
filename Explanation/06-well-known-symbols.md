# Слайд 6: Встроенные символы (well-known symbols)

## Что такое well-known symbols

Well-known symbols - это предопределенные символы, которые используются для переопределения поведения встроенных методов JavaScript.

## Основные well-known symbols

### Symbol.iterator
Используется для определения итерационного поведения объекта.

### Symbol.toPrimitive
Определяет поведение при преобразовании объекта в примитив.

### Symbol.toStringTag
Используется для настройки вывода Object.prototype.toString.call().

### Symbol.hasInstance
Используется для определения поведения оператора instanceof.

### Symbol.species
Используется для определения конструктора, который должен использоваться при создании производных объектов.

### Symbol.match / replace / search / split
Используются для переопределения поведения строковых методов.
